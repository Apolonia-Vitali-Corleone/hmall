# 项目说明

这是一个根据《黑马商城》为基础，逐步修改和优化的一个商城项目。

前端：hmall-nginx

后端：hmall



# hmall

## 服务模块

cart-service

item-service

pay-service

trade-service

user-service

## 网关模块

hm-gateway

## 通用模块

hm-common

hm-api



# hmall-nginx

在目录下,打开cmd执行：

```
start nginx.exe
```

前端项目启动完成，默认端口为18080
